# Contract: devboost.lock + config/mise.toml

## devboost.lock  (FR-005)
- location: repo root `devboost.lock`; format: lines `module<TAB>resolved-version`, sorted by module,
  trailing newline; committed; NO secrets. Regenerated by `lc_lock_write` (called by update + install).
- determinism: same resolved state → byte-identical file (sort -u). Tests: regenerate twice → identical;
  sorted; no secret-looking tokens.

## config/mise.toml  (FR-004)
- runtime pins file; `update` seeds it from resolved state if absent, else proposes bumps. Never committed
  automatically. Tests: seeded when absent; update writes proposals; no git commit.
